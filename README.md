# 🎮 Dora Game

A fun browser-based arcade game inspired by Flappy Bird, featuring Doraemon-themed characters, obstacles, and gadgets.

## 🚀 Features

- 🐦 Play as Doraemon
- 🚧 Avoid obstacles like Gian, Teacher, Mom, and Homework
- 🎁 Collect gadgets for bonus points
- 🎵 Sound effects and background music
- 🏆 High score system (saved using localStorage)
- 🔁 Restart functionality

---

## 🎮 How to Play

- Click or press **Space** to start
- Click or press **Space** to jump
- Avoid hitting obstacles
- Collect gadgets to earn extra points
- Try to beat your high score!

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
